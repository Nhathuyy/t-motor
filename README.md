# T-MOTOR — Thuê xe motor Đà Lạt

Landing page cho dịch vụ thuê xe máy tại Đà Lạt - chuẩn SEO, tối ưu Google AI Overview.

## Cài đặt

```bash
npm install
```

## Chạy

```bash
npm start
```

Mở trình duyệt: http://localhost:3000

## SEO

- Meta tags đầy đủ
- Schema.org (LocalBusiness + FAQPage) cho Google AI Overview
- Sitemap: `/sitemap.xml`
- Robots: `/robots.txt`

**Lưu ý:** Đổi `yourdomain.com` trong `index.html`, `sitemap.xml`, `robots.txt` và số điện thoại khi deploy.
